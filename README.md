# Housing Price Predictor
Work in progress - Finished on July 20, 2017

# What it does
My housing price predictor makes use of publically avaible real estate data in the Seattle area
The model is an implementation of XGBoost, will predicts the present and past values of real estate.
It gives the user insight into how to get the most value out of their property, and provides options for possible changes to the property, coupled with updated price predictions.
